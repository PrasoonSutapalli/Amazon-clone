# Amazon E-Commerce Clone

A high-fidelity, full-stack responsive e-commerce platform inspired by Amazon.  
This project demonstrates a complete user journey from account registration to dynamic product browsing, featuring a secure backend and an interactive frontend.

---

## Features

### User Authentication & Security

- Full-Stack Sign-up/Login system using PHP and MySQL  
- SQL Injection Prevention using MySQLi prepared statements  
- Regex-based client-side validation enforcing strong password policies  
  - Uppercase letters  
  - Lowercase letters  
  - Numbers  
  - Special characters  

---

### Dynamic Frontend

- Interactive JavaScript image slider with auto-transition and manual controls  
- ES6-based dynamic product rendering for “Today's Deals” section  
- Multi-level responsive sidebar navigation using Flexbox  
- Discount labels generated dynamically from data objects  

---

### UI/UX Design

- Amazon-inspired card-based layout  
- Navigation bar styled using CSS Grid and Flexbox  
- Integrated Google Fonts (Open Sans)  
- Font Awesome icons for professional styling

##  Installation & Setup

### 1️⃣ Clone the Repository
git clone https://github.com/PrasoonSutapalli/amazon-clone.git
### 2️⃣ Server Requirements

Install a local server environment such as:

- XAMPP  
- WAMP  

### 3️⃣ Database Setup

Create two databases:

- `login`
- `registeruser`

Create tables:

- `signin`
- `signup`

Ensure table structures match the PHP connection queries.

### 4️⃣ Run the Project

- Move project folder into `htdocs`
- Start Apache & MySQL in XAMPP
- Open browser:
- http://localhost/amazon-clone
